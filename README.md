# A-Share 5-Day Price Reversal Factor Research

> 基于中国 A 股 2010–2021 年日度行情数据，构建 5 日价格反转因子，完成因子检验、分层回测与约束二次规划（QP）组合优化。

---

## 项目简介

本项目为量化选股策略研究，核心流程包括：

1. **数据清洗**：剔除涨停、跌停、停牌、ST、新股等异常样本
2. **因子构建**：5 日价格动量因子，经 Rank 标准化与市值+行业中性化
3. **因子检验**：IC 检验、分层回测（8 层，等权 / 市值加权）
4. **组合优化**：约束二次规划（QP）均值-方差优化，带行业权重与个股权重约束

## 策略框架

### 1. 因子定义

```
factor = close_t / close_{t-5} - 1
```

- **5 日价格动量**：短期价格反转效应
- 每日截面对有效样本做 **Rank 标准化**（均值 ≈ 0，标准差 ≈ 1）

### 2. 中性化

- **方法**：市值（对数市值）+ 一级行业虚拟变量 OLS 回归取残差
- **输入**：Rank 标准化后的因子
- **输出**：`factor_rank_neu`

### 3. 收益率定义

- **持仓逻辑**：t+1 日开盘买入，t+2 日开盘卖出
- **收益率**：`return = open_{t+2} / open_{t+1} - 1`

### 4. IC 检验结果

| 指标 | Rank 原始 | Rank 中性化 |
|-----|----------|------------|
| IC 均值 | **-0.0580** | **-0.0575** |
| IC 标准差 | 0.1429 | 0.1134 |
| IR | -0.4059 | **-0.5070** |
| 年化 IR | -6.4441 | **-8.0479** |
| t 统计量 | -21.90 | **-27.35** |

> **信号方向**：因子值与下期收益负相关 —— 因子值越小，预期收益越高。

### 5. 分层回测（8 层）

- 按因子值每日截面分 8 层
- 测试方式：等权、市值加权
- Layer 1（因子值最小）→ Layer 8（因子值最大）
- 多空组合：Layer_1 - Layer_8

### 6. QP 组合优化（加分题）

#### 参数设置

```python
N_SELECT    = 100       # 持仓股票数
MIN_W       = 0.005     # 个股权重下限 0.5%
MAX_W       = 0.03      # 个股权重上限 3%
MAX_IND_W   = 0.20      # 行业最大暴露 20%
RISK_LAMBDA = 5.0       # 风险厌恶系数
REBAL_FREQ  = 1         # 调仓频率（日度）
COV_WINDOW  = 60        # 协方差估计窗口
```

#### 目标函数

$$
\max_w \quad \alpha^T w - \lambda \cdot w^T S w
$$

- `alpha = -factor_rank_neu`（反向取信号）
- `S`：样本协方差矩阵（年化，+ `1e-6 · I` 正则化）
- `lambda`：风险厌恶系数

#### 约束条件

| 约束 | 表达式 |
|-----|--------|
| 满仓 | $\sum w_i = 1$ |
| 个股权重 | $0.5\% \le w_i \le 3\%$ |
| 行业暴露 | $\sum_{i \in \text{行业}} w_i \le 20\%$ |

#### 选股流程

1. **预筛选**：按 alpha 取前 300 名候选
2. **行业容量**：每行业最多 40 只（`MAX_IND_W / MIN_W`），与 QP 约束自洽
3. **QP 求解**：`scipy.optimize.minimize` SLSQP，直接输出权重，无后处理清理
4. **日度调仓**

## 核心结论

- 5 日价格反转因子在 A 股市场具有显著的负向预测能力（IC ≈ -0.0575）
- 市值+行业中性化后，因子稳定性提升（IC 标准差从 0.1429 降至 0.1134）
- 分层回测中，Layer 1（低因子值组合）表现优于 Layer 8，多空组合具有正收益
- QP 组合优化通过样本协方差矩阵与严格约束，试图在控制波动的同时获取反转 Alpha
